## 🤖 AI / Prompt Engineer

### Уже сделано
- [x] Черновики промптов для генерации вопросов (устарели — убираем)
- [x] Базовая структура промпта для генерации результата

### В работе

#### Промпт 1 — Генерация промпта (ветка A)
```
Задача: собрать детальный промпт для мощной модели
        на основе структурированных данных пользователя
Вход:   тип, режим, наработки, задача, 5 ответов
Выход:  готовый промпт (только текст, без преамбулы)
Структура выхода: Роль → Контекст → Задача → Формат → Ограничения
```
- [ ] Написать и протестировать на 10+ разных входах
- [ ] Проверить качество для каждого типа (контент / код / отчёт)
- [ ] Убедиться что не «ломается» при пустых ответах
- [ ] Подобрать оптимальный max_tokens

#### Промпт 2 — Парсинг XML (ветка B2a)
```
Задача: разбить произвольный промпт на 6 смысловых блоков
Вход:   текст промпта
Выход:  JSON [{ "tag": "...", "content": "..." }]
Теги:   role, context, task, format, constraints, examples
```
- [ ] Проверить на промптах без явной структуры
- [ ] Проверить на промптах уже с XML-тегами
- [ ] Проверить на очень коротких промптах (1–2 предложения)
- [ ] Убедиться что JSON всегда валидный (добавить retry?)

#### Промпт 3 — Преобразование (ветка B2b)
```
Задача: изменить промпт согласно инструкции пользователя
Вход:   исходный промпт + инструкция (карточки + своё поле)
Выход:  изменённый промпт (только текст)
```
- [ ] Протестировать каждую карточку отдельно
- [ ] Протестировать комбинации карточек
- [ ] Протестировать своё поле с неожиданными инструкциями
- [ ] Убедиться что модель не «улучшает» то что не просили

### Бэклог — будущие промпты
- [ ] **Оценка качества промпта** — авто-скоринг (есть роль? есть формат? и т.д.)
- [ ] **Авто-подсказки** — если задача похожа на шаблон, предложить его

## MVP — критерии готовности

```
✅  Ветка A: флоу пройден → получен готовый промпт
✅  Ветка B2b: промпт вставлен → преобразован → скопирован
✅  Ветка B2a: промпт разбит → блоки редактируются → XML скопирован
✅  ResultScreen: копирование + ручное редактирование работают
✅  Ошибки API: не крашится, показывает понятное сообщение
✅  Десктоп: всё работает полностью
⬜  Мобилка: работает (кроме DnD в B2a)
```
