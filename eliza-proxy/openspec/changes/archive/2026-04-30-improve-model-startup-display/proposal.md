## Why

При запуске сервера модели отображаются с статусом ⏳, но никогда не обновляются до ✅/❌. Это происходит из-за двух основных проблем: 1) race condition между инициализацией данных моделей и callback-ами probe, и 2) отсутствие инкрементального обновления интерфейса по мере завершения тестов моделей. Кроме того, в backlog указаны дополнительные требования по улучшению отображения.

## What Changes

- Исправить race condition между инициализацией `modelsByProvider` и callback `onModelProbed`
- Реализовать инкрементальное обновление отображения по мере завершения probe каждого отдельного model
- Реализовать обновление progress bar для каждого провайдера по мере завершения моделей
- Реализовать последовательное отображение провайдеров с перезаписью строк (а не добавлением новых)
- Добавить общий progress bar для всех моделей в последней строке вывода
- Обеспечить корректное завершение тестов моделей с отображением финальных статусов
