## ADDED Requirements

### Requirement: Скачивание текущего изображения
Приложение SHALL позволять пользователю скачать текущее отображаемое изображение на устройство.

#### Scenario: Скачивание изображения
- **WHEN** изображение отображается и пользователь нажимает кнопку «Скачать»
- **THEN** браузер скачивает файл изображения с именем вида `picsum-<id>.jpg`

#### Scenario: Кнопка скачивания недоступна без изображения
- **WHEN** изображение ещё не сгенерировано
- **THEN** кнопка «Скачать» отсутствует или заблокирована (disabled)

#### Scenario: Ошибка скачивания (CORS)
- **WHEN** браузер блокирует fetch из-за CORS-ограничений
- **THEN** приложение показывает сообщение об ошибке и предлагает открыть изображение в новой вкладке
