## ADDED Requirements

### Requirement: Отображение случайного изображения
Приложение SHALL показывать случайное изображение из picsum.photos при нажатии кнопки «Генерировать».

#### Scenario: Первая загрузка страницы
- **WHEN** пользователь открывает `index.html` в браузере
- **THEN** приложение отображает интерфейс с кнопкой «Генерировать» и пустой областью для изображения

#### Scenario: Нажатие кнопки «Генерировать»
- **WHEN** пользователь нажимает кнопку «Генерировать»
- **THEN** в области изображения появляется новая случайная картинка размером не менее 600×400px

#### Scenario: Повторная генерация
- **WHEN** пользователь нажимает «Генерировать» повторно
- **THEN** загружается новое изображение, отличное от предыдущего

#### Scenario: Ошибка загрузки
- **WHEN** picsum.photos недоступен или изображение не загружается
- **THEN** приложение отображает текстовое сообщение об ошибке вместо сломанного изображения

### Requirement: Отображение метаданных изображения
Приложение SHALL показывать ID и размер текущего изображения под ним.

#### Scenario: Метаданные после генерации
- **WHEN** изображение успешно загружено
- **THEN** под изображением отображается его ID и разрешение (ширина × высота)
