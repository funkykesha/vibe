## Why

Нужен быстрый способ получить случайное изображение прямо в браузере без установки зависимостей. Одностраничное HTML-приложение решает это мгновенно.

## What Changes

- Новый файл `index.html` — самодостаточное приложение без внешних зависимостей
- Кнопка «Генерировать» загружает случайную картинку через публичный API (picsum.photos)
- Отображение изображения с указанием его ID и размера
- Кнопка сохранения/скачивания текущего изображения

## Capabilities

### New Capabilities

- `image-display`: отображение случайного изображения в браузере с кнопкой обновления
- `image-download`: скачивание текущего отображаемого изображения

### Modified Capabilities

## Impact

- Новый файл `index.html` в корне проекта (или отдельной папке `random-image-generator/`)
- Зависимость от внешнего API: picsum.photos (публичный, бесплатный)
- Нет серверных компонентов, нет сборки — чистый HTML/CSS/JS
